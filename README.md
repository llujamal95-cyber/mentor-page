# Mentor Portal: Login & Dashboard

A secure management web application designed for instructors, mentors, and advisors to track student performance, manage schedules, and provide academic guidance.

## 🚀 Features
* **Secure Instructor Authentication:** Private login portal with role-based access control to protect sensitive student records.
* **Mentor Dashboard:** A centralized control panel displaying daily schedules, pending tasks, and recent student submissions.
* **Student Progress Tracking:** Tools to view individual student analytics, attendance rates, grades, and historical performance data.
* **Session & Calendar Management:** An interactive schedule builder to book, update, or cancel mentorship sessions and office hours.
* **Feedback System:** Built-in tools to review assignments, leave detailed performance comments, and log internal student notes.
